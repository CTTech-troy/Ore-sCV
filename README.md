## üì¶ Installation & Setup

1. **Clone the repository:**

   ```bash
   git clone https://github.com/Mashaun18/Ore-sCV.git
   cd ore's CV/Frontend
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Run the development server:**

   ```bash
   npm run dev
   ```

4. **Build for production:**

   ```bash
   npm run build
   ```
